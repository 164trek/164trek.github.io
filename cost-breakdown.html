This is only the basic Gunung Tahan climbing cost. It is not the full Singapore-to-Malaysia trip cost.

How the cost is calculated

Shared group costs
Guide: RM1,500
4x4 trail transfer: RM300
Travel agent fee: RM300
Shared group total: RM2,100
Shared per pax for 10 pax: RM2,100 ÷ 10 = RM210 / pax

Per-person fees
Permit: RM1 / pax
Camping fee: RM4 / pax
Camera fee: RM5 / pax
Per pax fees total: RM10 / pax
Total basic climbing cost for 10 pax: RM210 + RM10 = RM220 / pax
Approx. SGD 75 / pax

Not included

Reservation note
